# OpenMoji Animal SVGs Setup

The animal illustrations use [OpenMoji](https://openmoji.org) (MIT/CC BY-SA 4.0 license).

## Download the required animals

Go to https://openmoji.org and download the SVG for each of these animals.
Save them to `public/animals/` with the exact filenames listed below:

| Filename | OpenMoji search term |
|---|---|
| `capybara.svg` | capybara |
| `axolotl.svg` | axolotl |
| `red-panda.svg` | red panda |
| `quokka.svg` | quokka |
| `hedgehog.svg` | hedgehog |
| `sloth.svg` | sloth |
| `chinchilla.svg` | chinchilla |
| `fennec.svg` | fox (use the fox emoji as fennec) |
| `meerkat.svg` | meerkat |
| `pygmy-owl.svg` | owl |
| `sugar-glider.svg` | flying squirrel (closest match) |
| `slow-loris.svg` | monkey or use a custom illustration |
| `binturong.svg` | bear (placeholder until found) |
| `fossa.svg` | cat (placeholder) |
| `kinkajou.svg` | raccoon (placeholder) |
| `patagonian-mara.svg` | rabbit (placeholder) |
| `numbat.svg` | anteater (placeholder) |
| `dik-dik.svg` | deer |
| `pika.svg` | chipmunk (placeholder) |
| `jerboa.svg` | mouse (placeholder) |

## Batch download script (optional)

You can also use the OpenMoji GitHub release to download all SVGs at once:
1. Visit https://github.com/hfg-gmuend/openmoji/releases
2. Download the latest `openmoji-svg-color.zip`
3. Extract and copy the relevant files to `public/animals/`

Note: OpenMoji uses emoji Unicode codepoints as filenames (e.g. `1F9A6.svg` for the otter).
You'll need to rename them to match the names above.
